# ☕ Teria Coffee – Sales Dashboard
An Excel dashboard analyzing coffee shop sales (2024–2025), using **Power Query** for data cleaning and **Pivot Tables** for insights.
## 🧰 Tools:
- Power Query
- Pivot Tables & Charts
- Slicers
## 📈 Highlights:
- 🥇 Top drink: Latte – $28K+
- ⏰ Peak hour: 10 AM
- 💳 96% card payments
- 📊 +29% sales growth in Q1 2025
